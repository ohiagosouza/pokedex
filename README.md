# Pokedex

- Pokedex App using PokeAPI to list pokemons
- Added a search bar to search for a pokemon by name

## 🚀 Starting

Instructions to get a copy

### 📋 Prerequisite

Knowledge in Javascript

### 🔧 Instalation

1. Clone

```bash
git clone https://github.com/ohiagosouza/pokedex.git
```

2. Go to directory

```bash
cd pokedex
```
